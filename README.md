## Run Locally

Clone the project

```bash
  git clone https://github.com/Upendrapal0607/ecommerce-app-8386.git
```

Go to the project directory

```bash
  cd ecomerce-app-destination
```

Install dependencies

```bash
  npm install 

```

<!-- Start the server

```bash
  npm run server 
``` -->
